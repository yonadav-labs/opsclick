What is an Environment

An Environment generally refers to a unique instance of the application that contains all underlying components necessary for the application run on a piece of hardware, VM or container. The application itself may sit within an environment, but it is important to understand the distinction between the two.

What is DevOps

Dev Ops is short for Development Operations. A person whose job is not to write code but to get it ‘released’, ‘deployed’ or ‘to automate application’s build and deployment processes’, is a DevOps person! What they do for you is DevOps in a nutshell!
